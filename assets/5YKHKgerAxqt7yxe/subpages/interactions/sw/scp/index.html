Create Your Self-Care Plan

How to Use This Tool

  1. Select a wellness dimension box.
  2. Review the suggestions in the dialog that appears.
  3. Click "Add Activity" to enter your activity description, date, and time.
  4. When finished, choose "Save & Close".
  5. Repeat for other dimensions as desired.
  6. Select "Generate My Self-Care Plan" to print your custom plan.
  7. Select "Add to Calendar" to download a single ICS file with all scheduled events.
